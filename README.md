# Super Samurai

A 3D wave-based survival game built with Three.js.

## Play

Open `index.html` in a browser to play.

## Features

- **8 Playable Characters**: Samurai, Knight, Shuriken Thrower, Assassin, Archer, Pikeman, Wizard, Necromancer
- **10 Themed Maps**: Forest, Frost, Swamp, Desert, Graveyard, Volcano, Crystal Cave, Sky Temple, Abyss, Dragon's Lair
- **50 Waves** of enemies with scaling difficulty
- **Boss Battles**: Unique bosses including the Dragon
- **25 Quests** with unique relic rewards
- **Skill Trees** for each character
- **Relic System** with passive effects
- **Upgrade Cards** between waves

## Controls

- **WASD** - Move
- **Mouse** - Look around
- **Left Click** - Attack
- **Right Click** - Parry (or shoot for Archer)
- **Space** - Dodge roll
- **Shift** - Sprint
- **Q/E** - Special abilities (Wizard/Necromancer)

## Characters

| Character | Role | Special |
|-----------|------|---------|
| Samurai | Balanced | Standard melee combat |
| Knight | Tank | High armor, block chance |
| Thrower | Ranged | Throws shurikens |
| Assassin | DPS | High crit, backstab bonus |
| Archer | Ranged | Bow attacks, dagger melee |
| Pikeman | Reach | Extended attack range |
| Wizard | Mage | Fireball & homing spells |
| Necromancer | Summoner | Summons skeleton minions |
